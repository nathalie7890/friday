# friday
(Semester 1 Project 2) Friday Dating
Online dating website that uses HTML, CSS, Bootstrap, JavaScript, PHP & SQL.

### Basic features include:
1. Register for an account and login.
2. Match users with mutual likes.
3. Real-time chat
